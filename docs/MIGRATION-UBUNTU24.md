## §5 — System Prerequisites (Ubuntu 24)

Run once on fresh VM:

```bash
# Base
sudo apt update && sudo apt install -y \
  curl git build-essential pkg-config \
  python3-dev python3-venv \
  libssl-dev libffi-dev

# Python 3.11 (Ubuntu 24 ships 3.12 by default — we need 3.11 for venv)
# Install via deadsnakes PPA or use uv's managed Python (recommended):
curl -LsSf https://astral.sh/uv/install.sh | sh
export PATH="$HOME/.local/bin:$PATH"
uv python install 3.11

# Docker Engine (NOT Docker Desktop — we're on native Linux now)
sudo apt install -y docker.io docker-compose-plugin
sudo usermod -aG docker $USER  # log out/in for group to take effect

# Node + Claude Code
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
npm install -g @anthropic-ai/claude-code

# GitHub CLI
sudo apt install -y gh

# Validation
python3 --version   # should be >= 3.11 available via uv
docker --version
docker compose version
claude --version
gh --version
```

**WSL2-specific quirks that go away on native VM:**
- Docker Desktop integration toggle — irrelevant; native Docker Engine now.
- `host.docker.internal` in [.env](/home/jfortin/openfleet/.env) for LightRAG → LocalAI — on native Linux this resolves differently. Options: run LocalAI inside the compose network, or set `LIGHTRAG_LLM_HOST=http://172.17.0.1:8090/v1` (docker bridge gateway), or add `extra_hosts: ["host.docker.internal:host-gateway"]` to the LightRAG service in `docker-compose.yaml`.

## §12 — Rollback Plan

If the target VM setup fails:

1. **Source machine untouched** — the WSL2 environment is not modified by rsync. Everything on source remains bootable.
2. **Partial setup on target** — `./setup.sh` is idempotent. Rerun after fixing the failing step.
3. **Docker state corruption** — `docker compose down -v && ./setup.sh` (WARNING: `-v` removes volumes; data is lost unless dumped).
4. **Venv corruption** — `rm -rf .venv && ./setup.sh` (recreates via `uv`).
5. **Vendor corruption** — `rm -rf vendor && ./setup.sh` (`scripts/install-openclaw.sh` reinstalls).
