# Fleet Evolution Vision

## Summary

Operator directives establishing the full fleet evolution beyond the tools system skills work. Covers agent directive chain revision, chain/bus architecture, brain evolution, RAG/knowledge systems, multi-model strategy, budget modes, agent lifecycle, signatures, and the autocomplete web/map net. These are the primary source of truth for all fleet evolution work.
